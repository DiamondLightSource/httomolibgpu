Methods Description#

Here we present a list of methods that exist in the HTTomolibGPU library with more detailed information in places compared to API reference.

Data correction module#

Modules from Data correction can be used as pre-processing (e.g. apply Outliers removal filter to raw projection data) or in some cases as post-processing (e.g. apply Median filter to the result of the reconstruction) tools.

  • Median filter
  • Outliers removal filter

Data denoising module#

Modules from Data denoising can be used as post-processing tools. For instance, denoising procedures can be applied to the results of the reconstruction.

  • Total Variation denoising (ROF)
  • Total Variation denoising (PD)

Stripes removal module#

Modules from Stripes removal normally used as pre-processing tools. Stripes removal is equivalent to removing ring artefacts in the reconstructed images.

  • Remove stripes by sorting
  • Remove stripes combo
  • Remove stripes Titarenko
  • Raven filter

Data rescale module#

Modules from Data rescale is usually needed when the data needs to be rescaled to be saved in different bit-type as images.

  • Rescale to integers
